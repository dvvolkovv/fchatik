## 📸 Как это выглядит

### 1. Кнопка в нормальном состоянии
```
┌───────────────────────────────────────┐
│  📎  🖼️  📄  🎨  │  🎤                │
│                                       │
└───────────────────────────────────────┘
        ↑ Серая кнопка микрофона
```

### 2. Во время записи
```
┌───────────────────────────────────────┐
│  📎  🖼️  📄  🎨  │  🔴  ← пульсирует │
│                                       │
│  💬 "Говорите..."                     │
└───────────────────────────────────────┘
```

### 3. Результат распознавания
```
┌───────────────────────────────────────┐
│  Привет, это тест голосового ввода    │
│  ▌                                    │
└───────────────────────────────────────┘
   ↑ Текст автоматически вставлен
```

## 🎨 Визуальные эффекты

### Анимация кнопки

```
Обычное состояние:
🎤  ← Серая, статичная

Начало записи:
🎤 → 🔴  ← Меняет цвет

Во время записи:
  🔴     🔴     🔴
   ↕      ↕      ↕
 (pulse) (pulse) (pulse)
 ← Постоянная пульсация
```

### Ripple эффект

```
     🔴
    ╱│╲
   ╱ │ ╲
  ╱  │  ╲
 ╱   │   ╲
← Волны расходятся от кнопки
```

## 🎓 Примеры использования

### Пример 1: Быстрый вопрос
```
👤 [🔴] "Какая сегодня погода в Москве?"
       ↓ (распознавание)
💬 "Какая сегодня погода в Москве?"
       ↓ (отправка)
🤖 "В Москве сегодня..."
```

### Пример 2: Длинный текст
```
👤 [🔴] "Напиши мне код для создания
        REST API на FastAPI с 
        поддержкой аутентификации"
       ↓
💬 Весь текст распознан
       ↓
🤖 Подробный ответ с кодом
```

### Пример 3: Продолжение
```
💬 "Привет, как дела?"
       ↓
👤 [🔴] "Расскажи про Python"
       ↓
💬 "Привет, как дела? Расскажи про Python"
   ↑ Добавлено к существующему тексту
```
